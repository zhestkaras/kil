Диаграмма АС ИС 
•	Процесс выполнения самостоятельной работ
•	Преподаватель предоставляет материалы к заданию (15 минут)
•	Преподаватель публикует задание в системе (5 минут)
•	Преподаватель объясняет задание (30 минут)
•	Преподаватель проверяет работу (12 часов)
•	Преподаватель выставляет оценку (15 минут)
•	Преподаватель отправляет комментарии студенту (30 минут)

•	Куратор уведомляет студентов о новом задании (10 минут)
•	Куратор следит за выполнением заданий

•	Студент ищет запись (15 минут)
•	Студент изучает материалы (6 часов)
•	Студент выполняет задание (20 часов)
•	Студент ищет преподавателя (30минут)
•	Сдает работу (15 минут)
•	Студент исправляет работу (при необходимости) (6 часов)
